# рЯУМ Overview

Resume Intelligence is an AI-powered career assistant built using **Python**, **Streamlit**, and **Google Gemini AI**.

It helps candidates improve their resumes, compare them with job descriptions, prepare for interviews, generate cover letters, rewrite resumes professionally, and receive AI-powered career guidance.

# вЪЩ Installation

Clone the repository

```bash
git clone https://github.com/abhi964869/Resume-Intelligence.git
```

Go inside the project

```bash
cd Resume-Intelligence
```

Create a virtual environment

```bash
python -m venv .venv
```

Activate it

Windows

```bash
.venv\Scripts\activate
```

Linux / Mac

```bash
source .venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```
